## When to use this skill:

- When creating new database models or entities
- When defining table schemas and column types
- When setting up model relationships (belongsTo, hasMany, etc.)
- When adding created_at/updated_at timestamps
- When defining foreign key constraints
- When adding database-level validations (NOT NULL, UNIQUE)
- When choosing appropriate data types for columns
- When working with Prisma schema files
- When defining TypeScript types for database entities
- When implementing cascade behaviors for relationships
