## When to use this skill:

- When adding Tailwind CSS classes to components
- When creating custom CSS utilities or classes
- When implementing design tokens (colors, spacing, fonts)
- When deciding between Tailwind utilities vs custom CSS
- When working with globals.css or CSS module files
- When configuring tailwind.config.js
- When maintaining consistent styling patterns
- When optimizing CSS for production (purging unused styles)
- When avoiding framework style overrides
- When styling responsive layouts
- When adding hover, focus, or active states
