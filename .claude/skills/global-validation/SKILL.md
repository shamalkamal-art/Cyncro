- When validating form inputs on client and server

- When implementing allowlist validation
- When checking data types, formats, and ranges
- When preventing SQL injection, XSS, or other attacks
- When validating business rules (dates, balances, etc.)
- When handling file upload validation
- When validating API request bodies
- When implementing consistent validation across endpoints
