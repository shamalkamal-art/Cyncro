## When to use this skill:

- When implementing mobile-first responsive designs
- When adding Tailwind breakpoint prefixes (sm:, md:, lg:, xl:)
- When creating fluid, percentage-based layouts
- When using rem/em units for scalability
- When sizing touch targets for mobile (min 44x44px)
- When testing UI across screen sizes
- When optimizing images for mobile
- When adjusting typography for readability on all devices
- When prioritizing content for smaller screens
- When writing media queries
- When building responsive navigation patterns
